# 集成回归测试检查清单

## 需求变更后必查项

当 AEB 或相关组件接口变更时，回归测试应覆盖以下检查点。

## 功能覆盖

- [ ] 正常触发路径：TTC <= 1.5、相对速度 > 0、sensor_valid=True。
- [ ] 阈值点 1.5 秒与相邻值 1.49 / 1.51。
- [ ] 相对速度为零与负值的不触发路径。
- [ ] 传感器无效时不触发制动。

## 异常与鲁棒性

- [ ] 负值 TTC、NaN、非数值输入。
- [ ] 缺失或类型错误的 sensor_valid。
- [ ] 多字段同时异常的边界组合。

## 执行与报告

- [ ] 所有新增用例可在隔离临时目录中 pytest 执行通过。
- [ ] 失败用例输出包含断言差异或异常类型。
- [ ] 测试报告引用所依据的规范文档来源。

## 演示验收对照

标准演示需求「TTC 小于等于 1.5 秒且相对速度为正时触发制动」应生成：

1. 至少一个阈值点用例（1.50）。
2. 至少一个阈值下侧用例（1.49）。
3. 至少一个阈值上侧用例（1.51）。
4. 相对速度非正用例。
5. 至少一个异常输入用例。
