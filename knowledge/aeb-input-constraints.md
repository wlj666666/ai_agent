# AEB 输入约束

## 概述

AEB（自动紧急制动）模块的输入必须满足以下物理与逻辑约束，测试用例应据此构造合法与非法输入。

## TTC（碰撞时间）

TTC 表示 Time-To-Collision，单位为秒。有效输入必须为非负有限数值。

- 当 TTC 小于等于 1.5 秒且满足其他触发条件时，系统应触发制动。
- TTC 等于 0 表示已碰撞或极近距，仍须按规范处理。
- TTC 为负值、NaN 或无穷大属于异常输入，不得作为正常触发路径。

## 相对速度

相对速度表示本车与前方目标的接近速度，单位为 m/s。

- 触发制动要求相对速度为**正**（目标正在接近）。
- 相对速度为零或负值时不应触发正常制动逻辑。
- 相对速度超出传感器量程时，应结合传感器有效性字段判断。

## 传感器有效性

传感器有效状态（sensor_valid）为布尔输入，表示测距与速度测量是否可信。

- sensor_valid 为 True 时，TTC 与相对速度方可参与制动判定。
- sensor_valid 为 False 时，不得触发制动，并应记录无效状态。
- 缺失 sensor_valid 字段视为异常输入。

## 组合触发条件

完整触发条件：TTC <= 1.5 秒 **且** 相对速度 > 0 **且** sensor_valid 为 True。

任一条件不满足时，预期结果为不触发制动或返回明确错误。
